# Stretchable Header View in Swift!

Implement stretchable header view in 3 easy steps!

<img src="https://github.com/alvinjohntandoc/StretchableHeaderView/blob/master/streatchable.gif?raw=tru" width="200">

1) Create a custom header view and conform to `HeaderViewStretchable`.
2) Set `StretchyViewController` as the parent class of your View Controller.
3) In your ViewController don't forget to pass the headerView and the scrollView by calling this func `setHeaderView(headerView, in: tableView)`
